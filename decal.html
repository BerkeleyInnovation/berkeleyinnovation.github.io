The Human-Centered {design.} DeCal

is a Berkeley Innovation-hosted, introductory class to design theory and practice, with a focus on human-centered design. Students learn about the design process and industry, practice using industry-standard tools, engage in hands-on activities during class and in homework assignments, and complete a midterm and a final design project.

The 2-unit P/NP course meets once a week, 6:30-8:30 PM on Thursdays in 310 Jacobs during the Fall 2017 semester. There are no prerequisites – this class is for anyone interested in learning about human-centered design. Check back soon for more information about the Spring 2018 offering of this class!
