The Human-Centered {design.} DeCal - Des Inv 98

is a Berkeley Innovation-hosted, introductory class to design theory and practice, with a focus on human-centered design. Students learn about the design process and industry, practice using industry-standard tools, engage in hands-on activities during class and in homework assignments, and complete a midterm and a final design project.

The 2-unit P/NP course meets once a week, 6:30-8:30 PM on Thursdays in 310 Jacobs during the Spring 2018 semester. There are no prerequisites – this class is for anyone interested in learning about human-centered design. Our first class of the Spring 2018 semester will be held on February 1, 2018. To enroll, you must attend the first class and there will be a lottery if demand exceeds the number of seats we have in the class. Check back soon for the Facebook event!
